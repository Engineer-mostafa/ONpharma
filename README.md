To Run App Install

Then Write In Terminal
----------------------------

-------------------------------------------------------------------------
    "scripts": {
        "test": "echo \"Error: no test specified\" && exit 1",
        "start": "npx nodemon app.js"
    },


Then Wirte In The Terminal
-----------------------------
    npm start
